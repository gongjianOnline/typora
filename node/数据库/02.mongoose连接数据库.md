 

## mongoose的使用

### 简介

是一个对文档模型(ODM)库,他对Node原生的mongoDB模块进行了进一步的优化封装,并提供了更多的功能.

### 优势

- 比Node远程的MongoDB驱动更容易

### 初始化

```javascript
npm init -y
npm install uniq mongoose
```
