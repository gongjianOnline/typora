# 10.Java文件类 File

## 构造方法

示例

```java
/*在指定路径下创建*/
File file01 = new File("./a.txt");
/*在父类路径名中创建*/
File file01 = new File("./","a.txt");
/*方式二*/
File file01 = new File("./");
File file02 = new File(file01,"a.txt")
```

## 创建文件及目录

```java
/*创建文件*/
Flie flie01 = new File("./a.txt");
file01.createnewFile(); /*返回 boolean ,true表示创建成功.false创建失败*/
/*创建目录*/
File file = new File("./a");
file.mkdir(); /*返回结果同上*/
/*创建多级目录*/
File file = new File("./a/b/c");
file.mkdirs();/*返回结果同上*/
```

## 判断和获取功能

```java
File f = new  File("./a.txt");
f.isDireCtory(); /*是否是目录*/
f.isFile(); /*是否是文件*/
f.exists(); /*路径是否存在*/
f.getAsolutePath(); /*获取绝对路径*/
f.getPath(); /*获取路径和文件名*/
f.getName(); /*获取文件名*/
f.List(); /*返回当前路径下所有的目录名*/
f.ListFiles();/*返回当前路径下所有的目录路径*/
f.delete(); /*删除路径*/
```

示例

```java
File f = new File(".\\");
System.out.println(Arrays.toString(f.listFiles()));
```
